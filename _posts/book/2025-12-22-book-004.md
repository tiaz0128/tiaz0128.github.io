## 리눅스, 클라우드 시대를 위한 준비

리눅스라고 하면 검은 화면에 흰 글씨만 잔뜩 있는 어두운 이미지를 떠올리는 분들이 많을 것입니다. 저 역시 처음 리눅스를 접했을 때는 **명령어 하나하나가 낯설고 막막**했습니다.

이 책은 그런 **리눅스 입문자들을 위한 친절한 안내서**{:.orange}입니다. '리눅스 기본기'라는 부제가 잘 맞는 입문서 입니다.

IT에서 리눅스는 피할 수 없는 커널(Kernel) 입니다. 게다가 요즘 대부분의 서비스는 클라우드 환경에서 운영됩니다. 책을 통해 **AWS(Amazon Web Services) EC2**까지 공부하고, 클라우드 시대에 꼭 필요한 역량을 준비하는 첫걸음으로 삼아보세요!

## 실습 중심의 구성

이 책의 가장 큰 장점은 **실습 위주의 구성**{:.yellow}입니다. 단순히 명령어를 나열하는 것이 아니라, 하나하나 따라 하면서 자연스럽게 익힐 수 있도록 구성되어 있습니다.

### 버추얼박스로 쉽게 시작하기

리눅스를 배우려면 먼저 리눅스를 설치해야 하는데, 이것부터 입문자에게는 큰 장벽입니다. 이 책은 버추얼박스를 이용하여 윈도우나 맥에서도 쉽게 리눅스 환경을 구축할 수 있도록 안내합니다.

### 우분투와 로키 모두 다루기

책에서는 데비안 계열의 **우분투**와 레드햇 계열의 **로키** 두 가지 배포판을 모두 다룹니다. 실무에서는 두 가지 계열의 리눅스를 모두 접할 수 있기 때문에, 미리 차이점을 익혀두면 큰 도움이 됩니다.

<div class="file-name">Ubuntu</div>

```bash
$ sudo apt update
$ sudo apt install nginx
```

<div class="file-name">Rocky Linux</div>

```bash
$ sudo dnf update
$ sudo dnf install nginx
```

## 단계별 학습 구성

책은 총 3개의 마당으로 구성되어 있습니다.

### 첫째마당: 리눅스 첫걸음

리눅스가 무엇인지, 어떻게 설치하는지, 기본 명령어는 어떤 것들이 있는지 등 **가장 기초적인 내용**을 다룹니다. 파일 시스템의 개념과 경로에 대한 이해도 이 마당에서 다룹니다.

```bash
# 절대 경로로 이동
$ cd /home/ubuntu/blog

# 상대 경로로 이동
$ cd ../
```

### 둘째마당: 리눅스 활용하기

기본을 익혔다면 이제 실무에서 자주 사용하는 기능들을 배울 차례입니다. 패키지 관리, 파일 압축, 프로세스 관리 등 **실무에 꼭 필요한 내용**들을 다룹니다.

특히 **셸 스크립트 프로그래밍** 챕터는 매우 유용했습니다. 반복적인 작업을 자동화할 수 있어서 생산성을 크게 높일 수 있습니다.

```bash
#!/bin/bash

cat << END
==========
Select a number ->
==========
1. Check disk usages
2. Check login users
------------------------
END
echo -n "Number : "
read number
case $number in
  1) df -h ;;
  2) who ;;
  *) echo "Wrong number" ;;
esac
```

### 셋째마당: 리눅스 실전 프로젝트

마지막으로는 **실전 프로젝트**를 통해 배운 내용을 종합적으로 활용합니다. 웹 서버 구축, 파일 서버 구축, 라즈베리파이 활용, 클라우드(AWS) 사용까지 다양한 실습을 제공합니다.

특히 SAMBA와 NFS 환경을 직접 구축하는 **파일 서버 만드는 실습**은 매우 실용적이었습니다.
