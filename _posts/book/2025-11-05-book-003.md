## 쿠버네티스 쉽게 시작하기

이 책은 기본적으로 **어느 정도 도커를 알고 있는 독자**를 대상으로 합니다. 하지만 환경 구축부터 쉽지 않은 게 쿠버네티스입니다.

책에서는 [**kind(Kubernetes in Docker)**](https://kind.sigs.k8s.io/){:.none target="_blank"}를 이용하여 로컬 환경에서 쿠버네티스 클러스터를 쉽게 구축하는 방법을 알려줍니다. Docker 컨테이너 안에서 쿠버네티스 노드를 실행하는 방식이라 별도의 VM 없이도 학습할 수 있어 편리했습니다.

### Mac/Windows에서 kind 설치하기

Mac/Windows 환경에서 kind를 설치하는 과정도 책에서 친절하게 안내해줍니다. 복잡한 설정 없이도 금방 따라할 수 있었습니다.

<div class="file-name">Linux/Mac</div>

```bash
$ go install sigs.k8s.io/kind@v0.30.0
```

<div class="file-name">Windows</div>

```bash
$ choco install kind
```

### kind로 클러스터 만들기

단 한줄로 쿠버네티스 실습 환경 구성 완료!

```bash
$ kind create cluster
```

## 귀여운 만화와 도해

이 책의 또 하나의 장점은 귀여운 만화와 도해입니다. 쿠버네티스 개념이 복잡하고 지루할 수 있는 지점마다 귀여운 일러스트와 만화가 나와서 자연스럽게 리프레시할 수 있었습니다.

단순히 귀엽기만 한 게 아니라, **심플하면서도 핵심을 찌르는 유익한 내용**으로 구성되어 있어 개념을 이해하는 데 큰 도움이 되었습니다. 복잡한 쿠버네티스 아키텍처를 시각적으로 쉽게 설명해주니 머리에 쏙쏙 들어왔습니다.

![귀여운 만화와 도해](/assets/img/content/book/003/004.webp){:.img-s}

`> 귀엽기만 한게 아니라 내용도 알차다!`{:.img-caption}

## 직접 만들고, 망가뜨리고, 고치며 배우기

이 책은 단순히 개념만 설명하는 것이 아니라, **직접 실습하면서 체득할 수 있도록**{:.orange} 구성되어 있습니다. Pod를 만들고, 의도적으로 장애 상황을 만들어보고, 다시 고치는 과정을 반복하면서 쿠버네티스의 동작 원리를 몸으로 익힐 수 있었습니다.

특히 '망가뜨리고 고치는' 과정이 인상적이었습니다. 실무에서 문제가 발생했을 때 어떻게 대응해야 하는지 미리 경험해볼 수 있어서 실용적이었습니다. 또한 [CKA(Certified Kubernetes Administrator)](https://training.linuxfoundation.org/certification/certified-kubernetes-administrator-cka/){:.none target="_blank"} 자격증 준비에도 많은 밑바탕이 된것 같습니다!
