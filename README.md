## Use docker
1. build image
```bash
$ docker build . -t gitblog
```
<br/>

2. Run docker server
```bash
$ docker run -d --rm --name gitblog -p 8080:4000 gitblog
```

<br/>

3. Stop docker
```bash
$ docker stop gitblog
```
